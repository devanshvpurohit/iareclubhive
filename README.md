# ClubHive

ClubHive is a secure and scalable college event management platform.

## Features

- **Dashboard**: Get an overview of all upcoming events and club activities.
- **Club Management**: Discover, join, and manage your favorite campus clubs.
- **Event Management**: Create and register for events with ease.
- **Secure Authentication**: Email and password-based login for all users.
- **Admin Controls**: Manage clubs, events, and generate reports.

## Technologies Used

- **Frontend**: Vite, React, TypeScript
- **Styling**: Tailwind CSS, shadcn/ui
- **Backend/Auth**: Supabase

## Getting Started

### Prerequisites

- Node.js & npm installed

### Installation

1. Clone the repository:
   ```sh
   git clone <YOUR_GIT_URL>
   ```

2. Navigate to the project directory:
   ```sh
   cd clubhive
   ```

3. Install dependencies:
   ```sh
   npm install
   ```

4. Start the development server:
   ```sh
   npm run dev
   ```
